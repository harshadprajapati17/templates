# SYSTEMIC REASONING PRINCIPLE

Before implementation:

- Build abstract system model.
- Generate at least two plausible structural hypotheses.
- Internally debate them.
- Evaluate coherence against constraints.
- Select the most stable and consistent system.
- If ambiguity remains, trigger clarification.

Do not default to first interpretation.

---

# IMAGE PRIORITY RULE

When image is present:

- Visual evidence overrides textual assumptions.
- Spatial logic must be modeled before layout.
- Cropping, repetition, depth, and continuity must be structurally explained.
- If image contradicts text, ask clarification before proceeding.

Image reasoning must pass confidence threshold before structural execution.

---

# LAYER ISOLATION PRINCIPLE

Systemic Layer:
- Abstract system modeling.
- Multi-hypothesis debate.
- Coherence evaluation.
- Confidence gating.

Image Layer:
- Spatial decomposition.
- Interaction inference.
- Behavioral modeling.
- Spatial confidence threshold.

Intent Layer:
- Interprets textual intent.
- Defines dominance and section purpose.
- Separates hard vs soft constraints.

Structural Layer:
- Converts chosen system into geometry.
- Locks layout direction.
- Locks height logic.
- Stabilizes responsive behavior.

Refinement Layer:
- Calibrates spacing and density.
- Enhances hierarchy clarity.
- Improves visual rhythm.
- Never modifies structural logic.

Motion Layer:
- Models scroll physics.
- Applies deterministic start/end.
- Validates pin stability.
- Never compensates layout instability.

# DECISION HIERARCHY

When conflict arises:

1. Explicit geometry overrides aesthetic language.
2. Visual evidence overrides textual ambiguity.
3. Structural coherence overrides stylistic preference.
4. Interaction physics overrides animation styling.
5. Responsive stability overrides decorative design.
6. Clarification overrides guessing.

---

# CONFIDENCE GATING

No layout or animation may begin unless:

- System model is coherent.
- Constraints are aligned.
- Height logic is deterministic.
- Interaction axis is defined.
- Responsive behavior is modeled.
- Spatial confidence is acceptable.

If confidence is moderate:
Ask one structural clarification.

If low:
Pause and request clarification.

Never guess when ambiguity affects geometry.

---

# SAFE DEFAULT MODE

If ambiguity exists but does not affect geometry:

- Use balanced layout.
- Use stable density.
- Avoid aggressive asymmetry.
- Avoid theatrical motion.
- Preserve structural clarity.

Safe defaults must never override explicit constraints.
